# 💰 Caixa v3.1 - Gestão de Transações

Sistema profissional de gestão de caixa para comércios locais (padarias, restaurantes, lojas).

## 🎯 Características

✅ **Troco Automático** - Calculado automaticamente (Dinheiro Recebido - Saídas)
✅ **Formas de Pagamento** - Dinheiro, Débito, Crédito, PIX
✅ **Relatórios Detalhados** - Por forma de pagamento
✅ **PDF Profissional** - Exporta relatório formatado
✅ **100% Offline** - Funciona sem internet
✅ **Instalável** - Como app nativo no celular
✅ **Backup Automático** - Salva todos os dados

## 🧮 Sistema de Troco v3.1

```
TROCO = Dinheiro Recebido - Saídas em Dinheiro

Exemplo:
Dinheiro vendido:      R$ 1.200
(-) Dinheiro gasto:    R$ 150
────────────────────────────
Troco para amanhã:     R$ 1.050

⚠️ Débito/Crédito/PIX não afetam
   (vão pra conta bancária)
```

## 💡 Como Usar o App

### Manhã (Abertura)
- Abre o app
- Vê o troco definido ontem
- Começa a vender

### Dia Todo
- Registra cada transação
- Seleciona forma de pagamento
- Sistema atualiza saldo em tempo real

### Noite (Fechamento)
1. Clica "Relatório"
2. Sistema sugere o troco automaticamente
3. Clica "Usar Isso"
4. Clica "PDF" (se quiser)
5. Pronto! Dia finalizado
